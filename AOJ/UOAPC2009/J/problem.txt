Problem J  Cash Register

手持ちの硬貨で P 円を支払う。このとき、支払う硬貨の枚数と
お釣りとして受け取る硬貨の枚数の合計の最小値を答えよ。

硬貨は 500円・100円・50円・10円・5円・1円の6種類がある。
お釣りは常に最小の枚数で返される。
手持ちの硬貨の総額は、P円以上であると仮定してよい。

Input
入力ファイルは、複数のデータセットから構成される。
一つのデータセットは、以下の形式で与えられる。

P N1 N5 N10 N50 N100 N500

Ni は、所持している i 円の硬貨の枚数を表す整数である。

P = 0 のとき、入力の終了を表す。このデータセットに対して出力をしてはならない。

Output
最小の硬貨の枚数を1行に出力せよ。

Sample Input
123 3 0 2 0 1 1
999 9 9 9 9 9 9
0 0 0 0 0 0 0

Output for Sample Input
6
3

Constraints
(データセットの数) <= 100
0 <= Ni <= 1000  for all i
